## [Finnish Embeddings](https://github.com/TurkuNLP/finnish-embeddings/tree/main)
The most recent one was work on [Finnish embeddings](https://github.com/TurkuNLP/finnish-embeddings/tree/main), where I...
- preprocessed raw-ish data into a new dataset
- embedded and indexed the data 
- run various tests on openly available embedding models
- visualized results
- annotated samples of the data

to name a few things. Not all work is openly accessible due to the restricted nature of the data.

## [OCR Post-Correction](https://github.com/TurkuNLP/ocr-postcorrection-lm/tree/main/evaluation)
The first project I got to contribute to was published as [OCR Error Post-Correction with LLMs in Historical Documents: No Free Lunches](https://aclanthology.org/2025.resourceful-1.8/) (Kanerva et al., 2025). In the [related repository](https://github.com/TurkuNLP/ocr-postcorrection-lm/tree/main/evaluation), some of my work can be accessed. Again, due to data distribution restrictions, not all my contributions are publicly available. My tasks included but were not limited to
- data preprocessing
- exploratory data analysis
- refactoring code
- annotation.
